These are the steps to push a github repository.

First lets discuss what is git?
Git is a free and open source distributed control system designed to handle
everything from small to very large projects with speed and efficiency.

Here are the steps to push your code to your own repository 


       
        1. Open Gitbash and clone the file

        2. Search the current folder using the following in the terminal.


           Output:
        
                     $ pwd
        

        3. List all the files   

            Output:
        
                    $ git add 'gitpush.html.py‘
        
        
        5. Commit the file to your local repository 

            Output:
        
                    $ git commit -m 'gitpush.html.py‘
        

        6. Push the changes to Github 

        Output:
         
            $ git push origin master
